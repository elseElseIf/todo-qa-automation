# Functional Requirements

## Task Management

- User can create a task.
- User can edit a task.
- User can delete a task.
- User can sort tasks by priority.
- User can find a task using the search menu.

## Data

- Tasks are saved after restarting the application.

## UI

- The task list is displayed correctly.
- Buttons are clickable.
- The application does not crash during normal use.
